# Gastro
Ich habe hier eine webbasierte Anwendung zur digitalen Essensplanung entwickelt. Mitarbeitende können sich registrieren und ihre Essenswahl eintragen. Der Koch als Administrator plant Gerichte und sieht Bestellungen ein. Das System reduziert Aufwand, vermeidet Lebensmittelverschwendung und organisiert die Planung effizient.
