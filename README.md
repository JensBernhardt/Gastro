# Gastro
Ich habe hier eine webbasierte Anwendung zur digitalen Essensplanung entwickelt. Mitarbeitende können sich registrieren und ihre Essenswahl eintragen. Der Koch als Administrator plant Gerichte und sieht Bestellungen ein. Das System reduziert Aufwand, vermeidet Lebensmittelverschwendung und organisiert die Planung effizient. Diese Version hat einen Fertigungsstand von 80 %. Für die Prüfung war sie jedoch zu 100 % fertig; es hätten lediglich noch einige CSS-Anpassungen vorgenommen werden müssen.
